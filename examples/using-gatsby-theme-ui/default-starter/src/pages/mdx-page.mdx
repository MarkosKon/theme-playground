## Import component

### Box + Button

Let's import and render a `Button` inside a `Box` component:

import Button from "@affectionatedoor/gatsby-theme-ui/src/components/Button";
import { Box } from "@affectionatedoor/gatsby-theme-ui/src/components/EmotionInMDX";

<Box sx={{ p: 4, bg: "mute" }}>
  <Button sx={{ bg: "secondary", color: "white" }}>Click me</Button>
</Box>

This is the source code:

```jsx
import Button from "@affectionatedoor/gatsby-theme-ui/src/components/Button";
import { Box } from "theme-ui";

<Box sx={{ p: 4, bg: "mute" }}>
  <Button sx={{ bg: "secondary", color: "white" }}>Click me</Button>
</Box>;
```

### Centered + Link

Let's import and render a `Link` component inside a `Centered` component:

import Link from "@affectionatedoor/gatsby-theme-ui/src/components/Link";
import Centered from "@affectionatedoor/gatsby-theme-ui/src/components/Centered";

<Centered sx={{ p: 3, bg: "mute", color: "bg" }}>
  <Link to="/" sx={{ p: 2 }}>
    Go to homepage
  </Link>
</Centered>
